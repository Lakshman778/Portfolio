Portfolio
Overview

This is Lakshman Kumar's Personal Portfolio Website, built using React.js. The website showcases my projects, skills, experience, and contact information in a clean and interactive format. It is designed to provide an engaging experience for recruiters, collaborators, and visitors.

Features

Responsive Design: Works on desktop, tablet, and mobile devices.

Project Showcase: Highlights all projects with descriptions, images, and links.

Skills Section: Displays technical skills with icons for better readability.

Experience & Education: Summarizes professional experience and educational background.

Contact Form: Lets visitors reach out directly via email.

Social Links: Includes LinkedIn, GitHub, and other social media links.

Technologies Used

Frontend: React.js, HTML5, CSS3, JavaScript

Icons: Lucide-React

Hosting: Netlify

Installation

To run the project locally:

Clone the repository:

git clone https://github.com/Lakshman778/Portfolio.git


Navigate to the project folder:

cd Portfolio


Install dependencies:

npm install


Start the development server:

npm start


Open in your browser:

http://localhost:3000
